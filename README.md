# Coproject – Группа III

## Структура проекта

### Parallelepiped.h – Базовый класс (Шаблон)
Определить в базовом классе:
− конструктор по умолчанию,
− конструктор с параметрами,
− деструктор,
− функции **set** и **get** для задания и получения значений ребер,
− функцию **show** для вывода ребер на экран,
− функции для изменения ребер,
− функцию вычисления площади поверхности,
− функцию вычисления объема.

---

### Kub.h – Производный класс Куб
Определить в классе наследнике:
− перегруженную функцию для изменения ребер,
− функцию вычисления радиуса описанной окружности,
− перегруженную функцию вычисления площади поверхности,
− перегруженную функцию вычисления объема куба.

---

### Kvadrat.h – Производный класс Квадрат
Создать производный класс Квадрат от класса Kub (куб). Определить в классе наследнике:
− перегруженную функцию для изменения сторон,
− функцию вычисления радиуса описанной окружности,
− перегруженную функцию вычисления площади.

---

### Rectangle.h – Производный класс Прямоугольник
Определить в классе наследнике:
− перегруженную функцию для изменения сторон,
− функцию вычисления радиуса описанной окружности,
− перегруженную функцию вычисления площади,
− функцию для нахождения периметра.
